**Transparenz schafft Handlungsfähigkeit.** 

Dieses Dashboard ist ein Prototyp für eine neue Form der Bürgerbeteiligung in Villingen-Schwenningen. In Zeiten der Haushaltskrise dürfen Entscheidungen nicht hinter verschlossenen Türen fallen. 

### 🚀 Was bietet die VS-Waage?
* **Echtzeit-Simulation:** Wie wirken sich Sparmaßnahmen auf den "Motor" der Stadt aus?
* **Transparenz:** Visualisierung der Finanzströme zwischen den Stadtbezirken.
* **Bürger-Check:** Interaktive Szenarien statt 600-seitiger Haushalts-PDFs.

### 🛠 Technik & Vision
Basierend auf **Python**, **Streamlit** und **Plotly**. Dieses Tool nutzt Open-Source-Technologie, um Verwaltungsprozesse effizienter und für jeden Bürger greifbar zu machen.
